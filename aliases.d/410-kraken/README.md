# 410-kraken

Atajos para trabajar con kraken-node y el workspace de pruebas automáticas.

## Funciones
- kraken_install: instala kraken-node globalmente, genera un esqueleto y agrega dependencias locales.
- kraken_run: ejecuta 
pm run test --workspace=e2e/misw-4103-kraken.
- kraken_reports_clear: elimina los reportes históricos (/e2e/misw-4103-kraken/reports/).
- chromium_listen: levanta chromedriver en el puerto 4444 para conectar navegadores.
